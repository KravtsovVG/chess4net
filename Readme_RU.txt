Дебют
--------

Chess4Net - это программа для игры в шахматы по Интернету. Она может быть использована как самостоятельное приложение (сокет-версия для Microsoft Windows или Linux), либо в качестве плагина для таких интернет-пейджеров как Miranda, QIP Infium, Trillian и &RQ. Поддерживаются две разновидности шахмат: классические и вариант случайных шахмат PP Random Chess. Chess4Net поддерживает контроль времени "внезапная смерть" и контроль с инкрементацией времени. Есть также возможность дать партнёру гандикап по времени. Если Вы захотите улучшить своё мастерство, то можно воспользоваться режимом игры с тренировкой.

Митеншпиль
------------

Установка:

Сокет-версия:
  Разархивируйте архив в любую папку на своём компьютере.

Плагин для Miranda:
  Разархивируйте архив в директорию плагинов (Plugins) для ИП Miranda и (пере-)запустите его.

Плагин для QIP Infium:
  Разархивируйте архив в директорию плагинов (Plugins) для ИП QIP Infium и (пере-)запустите его.

Плагин для Trillian Pro:
  Разархивируйте архив в директорию плагинов ИП Trillian Pro; зайдите в Trillian->Trillian Preferences..., выберите Plugins и отметьте галочкой Chess4Net_trillian.dll

Плагин для &RQ:
  Разархивируйте папку архива Plugins\ в инсталяционную директорию &RQ (та, в которой находится &RQ.exe)

Как играть:

Сокет-версия:
  1) Запустите Chess4Net.exe
  2) Выберите 'Connect...' ('Подсоединиться...')
  3) Введите свое имя в поле 'Your Nick' ('Ваш ник').
    а) Выберите Server (сервер); сообщите вашему партнёру ваш IP (Чтобы определить IP под Windows, введите 'ipconfig' в Start->Run (Пуск->Запустить...) или в командной строке; Под Linux запустите ./get_ip.pl из терминала); введите № порта (согласуйте его с вашим партнёром).
    б) Выберите Client (клиент); Введите IP или доменное имя, сообщённое вашим партнёром; введите номер порта в Port# (№ порта) (каждая инстанция Chess4Net требуюет своего уникального значения).
  4) Нажмите OK и дождитесь завершения соединения.

Миранда:
  1) Кликнете правой кнопкой мыши на контакт, с которым вы хотите сыграть в шахматы и выберите 'Chess4Net' из контекстного меню.
  2) Дождитесь завершения соединения.

QIP Infium:
  1) Откройте окно сообщений контакта, с которым вы хотите сыграть в шахматы.
  2) Нажмите на кнопку 'Chess4Net' (под верхним фреймом справа, где отображаются входящие и исходящие сообщения).

Trillian Pro:
  1) Кликнете правой кнопкой мыши на контакт, с которым вы хотите сыграть в шахматы и выберите 'Chess4Net' из контекстного меню, либо откройте окно сообщений и наберите без кавычек "/CHESS4NET".
  2) Дождитесь завершения соединения.
  Для заметки: Chess4Net использует для передачи ходов и др. информации текстовые сообщения. В Trillian Pro отсутствует возможность подавления входящих сообщений, наподобие того, как в ИП Miranda. Поэтому имеет смысл отключить звук на входящие сообщения.

&RQ:
  1) Откройте окно сообщений контакта, с которым вы хотите сыграть в шахматы.
  2) Нажмите кнопку 'Chess4Net' (под верхним фреймом, где отображаются входящие и исходящие сообщения).
  3) Дождитесь завершения соединения.

Все настройки в Chess4Net выполняются через контекстное меню, поэтому, если вы не знаете, что делать дальше, не думайте, сразу же нажимайте правую кнопку мыши.
После нескольких запусков в инсталляционной директории (сокет-версия) или в папке \Chess4Net (интернет-пейджеры) появляется файл Chess4Net_GAMELOG.txt со всеми сыгранными в Chess4Net партиями, пользовательская база ходов (Chess4Net.pos и Chess4Net.mov) и внешние базы ходов (другие файлы с расширениями pos и mov), используемые в тренировочном режиме.

Тренировочный режим:

Тренировочный режим можно включать/выключать в меню Game Options (Опции игры). Пользовательская и внешние базы ходов используются только в тренировочном режиме. Пользовательская БХ содержит все партию, сыгранные пользователем в тренировочном режиме, и данные из неё используются для генерации подсказок с приоритетами в новых тренировочных сеансах. Пользовательская БХ может "самообучаться" на основе сыгранных партий, но также вы можете подключать внешние базы для улучшения качества порождаемых подсказок.
